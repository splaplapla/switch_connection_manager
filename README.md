# SwitchConnectionManager
* Switchとプロコンの接続を管理します

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'switch_connection_manager'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install switch_connection_manager
